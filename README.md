# 🚀 GlassPulse Dashboard - 5 Day Challenge

A modern, sleek, and highly interactive Personal Dashboard inspired by the **Glassmorphism** design trend. This project was built from scratch as part of a 5-day intensive coding challenge to master Front-end development.

## ✨ Key Features
- **Modern UI:** Built using CSS Grid and Flexbox for a clean, structural layout.
- **Glassmorphism:** Uses `backdrop-filter` and transparency for a premium frosted-glass look.
- **Dynamic Greeting:** Personalised greetings (Morning/Afternoon/Evening) based on system time.
- **Smart To-Do List:** 
  - Add/Delete tasks with category tags.
  - **Local Storage:** Your data persists even after refreshing the page!
  - Strike-through animation for completed tasks.
- **Theme Switcher:** Seamlessly toggle between **Dark Mode** and **Light Mode**.
- **Live Search:** Quickly filter through your tasks in real-time.
- **Fully Responsive:** Optimised for all devices (Mobile, Tablet, and Desktop).

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Grid Layout, Animations, Media Queries, and Glassmorphism.
- **Vanilla JavaScript:** DOM Manipulation, Local Storage API, and Event Listeners.
